Introduction

The Cerulean Necktie is an esolang (esoteric programming language) created by me just for fun and minimalistic programming. Why the name "necktie"? The logo for the portfolio ("The Husky") wears one.

Esolang is basically a programming language made for various purposes which can be programatically useful or useless. If it's "useful" the language is said to be Turing-Complete, meaning that it can solve computational problems and build applications. If it's "useless", it's for recreational purposes.

Syntax

To print a word or sentence, type :w: ('write') followed by a sentence:

:w: Hello world.

Output:

Hello world.

If a gibberish is typed:

Pineapple on pizza is not a pizza!

Output (it roasts the user):

What an unparliamentary language!

To repeat numbers until it reaches a certain number (ex. for loop in range(x,y) in Python), type :l: ('loop') followed by a variable of choice, ->R ('up to a range'), and a number ('range(5)'):

:l: var ->R 5

Output:

0 1 2 3 4

Lastly, the "fizz buzz" game. Any number between 1 and 21 that is either divisible by 3 or 5 should output "fizz buzz", whereas "fizz" is by 3 and "buzz" is by 5. & specifies 'and' and modulo specifies the modulo operation.

:l: var ->R 1 & 21 modulo 3 & 5

Output:

{3 - Fizz} {5 - Buzz} {6 - Fizz} {9 - Fizz} {10 - Buzz} {12 - Fizz} {15 - Fizz Buzz} {18 - Fizz} {20 - Buzz}
